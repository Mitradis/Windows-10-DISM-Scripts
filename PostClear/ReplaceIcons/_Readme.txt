Это простой и эффективный набор для замены всех иконок в ОС на иконки с Windows 7.
Для применения просто запускать _Replace.bat пока в папке не останутся .mun файлы.
После необходимо запустить WinTool и сделать сброс папок - это очистит кеш ОС.

Предполагается, что набор применяется в чистой системе, в условиях полного
отсутствия стороннего софта. В противном случае любые программы могут блокировать
эти файлы, что сделает их замену физически невозможной. Если система не чистая
закройте абсолютно все приложения в диспетчере задач самостоятельно.
=========================================================================
This is a simple and effective set for replacing all icons in the OS with icons from Windows 7.
To use, simply run _Replace.bat until there are no .mun files left in the folder.
Afterwards you need to run WinTool and reset the folders - this will clear the OS cache.

It is assumed that the kit is used in a clean system, under conditions of complete
lack of third-party software. Otherwise, any programs may block
these files, making them physically impossible to replace. If the system is not clean
close absolutely all applications in the task manager yourself.
