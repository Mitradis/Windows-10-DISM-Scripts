Данная версия Windows 10 представляет из себя глубоко облегченную версию стандартного образа.

При необходимости настройки VPN подключения, установите режим запуска службы "Диспетчер подключений удаленного доступа" на "Вручную".

Полный сброс правил групповых политик:

RMDIR /S /Q "%WinDir%\System32\GroupPolicyUsers"
RMDIR /S /Q "%WinDir%\System32\GroupPolicy"
gpupdate /force

Установка приложений из Microsoft Store без приложения Магазин:

Найти нужное приложение на официальной странице Microsoft Store и скопировать ссылку на него;
Зайти на сайт https://store.rg-adguard.net и вставить и поле ссылку на нужное приложение;
Скачать все пакеты для целевой системы (пакеты x86/x64 и основной пакет приложения);
Установить сначала зависимые пакеты, а потом основной пакет приложения.
Если установка пакетов не начинается или завершается с ошибкой, можно попробовать установить их командой в PowerShell: Add-AppxPackage -Path "путь к пакету"

Приложения могут не запустится если нет подключения к интернету.

Описание функций WinTool:

  • Сервис:
    • Перезапустить проводник - закрытие и запуск проводника по команде.
    • Сбросить папки - очищает ключи реестра хранящие параметры отображения для каждой открытой когда либо папки.
    • Сбросить микшер - очищает ключи реестра настроек микшера хранящие параметры громкости всех когда либо запущенных приложений. Может помочь при проблемах воспроизведения в приложениях.
    • Сбросить совместимость - удаляет ключи реестра хранящие настройки совместимости когда либо выставленных для приложений.
    • Сбросить точки подключения - выполняет системную команду mountvol удаления всех не подключенных точек подключения томов. Всем вновь подключенным внешним посителям будут выданы буквы так, как будто они были подключены впервые.
    • Сбросить историю - удаляет историю запуска приложений в меню Пуск.
    • IPconfig flushDNS - сбрасывает кеш DNS.
    • UAC - переключение контроля учетных записей.
    • Обновление сертификатов - обновление корневых сертификатов с файла roots.sst на рабочем столе или скачивание его с официальных серверов.
    • Брандмауэр Windows - требуется для работы Служба установки Store.
    • Служба установки Store - поддержка установки .appx пакетов (в т.ч. драйверов), может быть отключена после конечной настройки компьютера.
    • Задача System Restore - возможность удаления автоматически создаваемой задачи создания контрольных точек восстановления. Задача создается при открытия меню восстановления системы.
    • Edge Update\EdgeWebView - переключение блокировки функции Edge\Edge Update\EdgeWebView. Удаление приложения Edge Update и установка максимальной версии Edge\Edge Update\EdgeWebView при блокировке.
    • Скриншот по Win+Shift+S - переключение функции создания избирательного снимка экрана.
    • Стандартный Пуск - переключение функционирования стандартного меню Пуск оболочки системы.
  • Пункты контекстного меню - переключение пунктов контекстного меню которые отключаются по умолчанию.
  • Этот компьютер - переключение отображения элементов в окне Этот компьютер.

Ссылки автора: Автор, Windows-10-DISM-Scripts, DISM-Scripts-Tools. Сторонние: superUser, ViVe, ClassicShell, LGPO.
